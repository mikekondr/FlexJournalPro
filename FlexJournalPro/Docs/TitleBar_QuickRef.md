## ?? Кнопки управління

### Згорнути (Minimize)
- **Іконка:** `WindowMinimize`
- **Hover:** Білий фон 10%
- **Action:** Згортає вікно у панель завдань

### Розгорнути/Відновити (Maximize/Restore)
- **Іконка:** `WindowMaximize` ? `WindowRestore` (динамічна зміна)
- **Hover:** Білий фон 10%
- **Action:** Максимізує або відновлює вікно

### Закрити (Close)
- **Іконка:** `Close`
- **Hover:** **Червоний фон (#E81123)**
- **Pressed:** Темно-червоний (#8B0A14)
- **Action:** Закриває програму

## ??? Технічні особливості

### WindowStyle
- **Значення:** `None`
- **Причина:** Прибрати стандартний заголовок

### WindowChrome
- **CaptionHeight:** 48px (висота для drag)
- **ResizeBorderThickness:** 5px (товщина для resize)
- **UseAeroCaptionButtons:** False (кастомні кнопки)

### Розміри
- **Висота заголовку:** 48px
- **Логотип:** 28x28px
- **Назва:** 16px (SemiBold)
- **Підзаголовок:** 12px (70% opacity)
- **Іконки кнопок:** 16x16px
- **Кнопки:** 46x32px

## ?? Використання

Заголовок автоматично працює після запуску програми. Ніяких додаткових налаштувань не потрібно!

### Якщо потрібно змінити текст:
```csharp
// У MainWindow.xaml
<TextBlock Text="Ваша назва"/>
<TextBlock Text="Ваш підзаголовок"/>
```

### Якщо потрібно змінити логотип:
```xml
<materialDesign:PackIcon Kind="ВашаІконка" Width="28" Height="28"/>
```

## ?? Примітки

- Заголовок автоматично адаптується під розмір вікна
- Tooltips показуються на всіх кнопках
- Іконка maximize/restore змінюється автоматично
- Працює на всіх підтримуваних версіях Windows
